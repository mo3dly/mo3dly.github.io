سياسة الخصوصية

تاريخ النفاذ: 2025-3-1

١. نطاق السياسة

تشرح هذه السياسة كيفية تعامل موقع "معدلي" مع بيانات الزوار، مع الالتزام بالشفافية وحماية الخصوصية وفقًا لأفضل الممارسات.

٢. البيانات المُجمَّعة

أ. بيانات التحليل (Google Analytics)

  • نستخدم إحصاءات Google لجمع معلومات مثل:
    • عنوان IP (لأغراض جغرافية)
    • نوع المتصفح ونظام التشغيل
    • الصفحات المُزارة ومدة الزيارة

ب. بيانات الإعلانات (Adsterra)

  • تعرض المنصة إعلانات مُوجَّهة بناءً على:
    • الموقع الجغرافي
    • نوع الجهاز (جوال/كمبيوتر)
  • تُدار هذه البيانات عبر سياسة خصوصية Adsterra.

٣. كيفية استخدام البيانات

  • تحسين تجربة المستخدم عبر تحليل السلوكيات.
  • تشغيل الإعلانات وعرضها بشكل فعَّال.
  • ضمان الاستضافة الآمنة عبر GitHub Pages.

٤. خدمات خارجية

  • Google Analytics: سياسة الخصوصية
  • Adsterra: سياسة الخصوصية
  • GitHub Pages: سياسة الخصوصية

٥. ملفات تعريف الارتباط (Cookies)

قد نستخدم ملفات كوكيز لأغراض تقنية أو تحليلية، ويمكنك إدارتها عبر إعدادات المتصفح.

٦. حقوق المستخدم

  • طلب حذف البيانات غير الضرورية.
  • تعطيل التتبع عبر إضافة Google Analytics Opt-out.

سيتم نشر أي تغييرات على هذه السياسة مع تحديث تاريخ النفاذ أعلاه.
